🏢 Company Visitors Management System (CVMS)

A smarter way to manage your office visitors ✨
Built to simplify entry, enhance security, and digitize the entire visitor management process.

🚀 Project Overview

Welcome to the Company Visitors Management System (CVMS) — a modern web-based solution to streamline how companies handle visitor check-ins and check-outs. Say goodbye to paper registers and manual entries!

Whether you're a receptionist, an admin, or security personnel, CVMS brings real-time visibility, instant notifications, and a professional front-desk experience to your organization.

🔑 Key Features

✅ Secure Visitor Registration (with photo, ID, and purpose)

📇 Employee Directory Integration

🕒 Check-In / Check-Out Tracking

📧 Instant Notifications to Hosts

📊 Daily/Monthly Reports

🔐 Admin & Receptionist Panels

📁 Database-Driven (MySQL / MongoDB)

🚶‍♂️ Getting Started

Prerequisites

XAMPP / WAMP (for PHP version)

Node.js (for JS version)

MySQL / MongoDB

Code Editor (e.g. VS Code)

Steps to Run

1. Clone the repo
2. Set up your database (import schema.sql)
3. Configure database credentials
4. Run the server (Apache/Node)
5. Visit: http://localhost/cvms
